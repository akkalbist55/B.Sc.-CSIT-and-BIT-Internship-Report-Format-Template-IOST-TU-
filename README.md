# B.Sc.-CSIT-and-BIT-Internship-Report-Format-Template-IOST-TU-

This repository contains a professionally structured internship report format template for students of **Tribhuvan University, Institute of Science and Technology (IOST)** pursuing **B.Sc. CSIT** and **BIT** programs.  
The template is intended for the **VIII Semester Internship Report** and follows commonly accepted academic formatting standards.
